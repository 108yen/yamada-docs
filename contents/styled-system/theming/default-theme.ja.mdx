テーマのトークンは、カテゴリ化されています。

:::note
テーマをカスタマイズしたい場合は、[テーマをカスタマイズする](/docs/styled-system/theming/customize-theme)をご覧ください。
:::

## Colors

`color`・`borderColor`・`backgroundColor`・`fill`・`stroke`などは、このトークンを参照されます。

:::note status=warning
各コンポーネントは、`colorScheme`という概念があります。`colorScheme`は、このトークンを元に生成されます。生成には条件があり、キーが`brand`・`primary`・`secondary`・`warning`・`danger`・`link`か、`Hue(50から950)`が定義されている場合に生成されます。
:::

## Transitions

- `transitionProperty`は、この`transitions.property`を参照されます。
- `transitionDuration`・`animationDuration`は、この`transitions.duration`を参照されます。
- `transitionTimingFunction`・`animationTimingFunction`は、この`transitions.easing`を参照されます。

```ts live=false
import { extendTheme } from '@yamada-ui/react'

const transitions = {
  property: {
    common: 'background-color, border-color, color, fill, stroke, opacity, box-shadow, transform',
    colors: 'background-color, border-color, color, fill, stroke',
    dimensions: 'width, height',
    position: 'left, right, top, bottom',
    background: 'background-color, background-image, background-position',
  },

  easing: {
    'ease-in': 'cubic-bezier(0.4, 0, 1, 1)',
    'ease-out': 'cubic-bezier(0, 0, 0.2, 1)',
    'ease-in-out': 'cubic-bezier(0.4, 0, 0.2, 1)',
  },

  duration: {
    'ultra-fast': '50ms',
    faster: '100ms',
    fast: '150ms',
    normal: '200ms',
    slow: '300ms',
    slower: '400ms',
    'ultra-slow': '500ms',
  },
}

const theme = extendTheme({ transitions })
```

## Breakpoint

