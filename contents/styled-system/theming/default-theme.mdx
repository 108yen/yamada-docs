## Colors

`color`, `borderColor`, `backgroundColor`, `fill`, `stroke`, etc., refer to this token.

:::note status=warning
Each component has a concept called `colorScheme`. `colorScheme` is generated based on this token. There are conditions for generation, and it is generated when `Hue (from 50 to 950)` is defined.
:::

## Transitions
