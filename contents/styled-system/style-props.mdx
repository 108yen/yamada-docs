`Style props` is a method to change the style of a component just by passing `props` to the component. It also provides many useful shorthands, improving development efficiency.

```tsx
<Box w="full" p="md" bg="warning" color="white">
  This is Box
</Box>
```

## Color and Background

```tsx live=false
// When to use theme token value.
<Box color='primary' />

// When to use custom value.
<Box bg='#FFF' />

// When to use responsive styles.
<Box color={{ base: 'primary', md: 'danger' }} />

// When to use color mode styles.
<Box bg={['white', 'black']} />
```

```tsx live=false
// When to use custom value.
<Box filter='grayscale(80%)' />

// When to use theme token value.
<Box filter='auto' blur='md' />
```

| Prop                 | CSS Property      | Theme Tokens                                            |
| -------------------- | ----------------- | ------------------------------------------------------- |
| `filter`             | `filter`          | none                                                    |
| `blur`               | `filter`          | [blurs](/styled-system/theming/default-theme#blurs)     |
| `brightness`         | `filter`          | none                                                    |
| `contrast`           | `filter`          | none                                                    |
| `dropShadow`         | `filter`          | [shadows](/styled-system/theming/default-theme#shadows) |
| `grayscale`          | `filter`          | none                                                    |
| `hueRotate`          | `filter`          | none                                                    |
| `invert`             | `filter`          | none                                                    |
| `saturate`           | `filter`          | none                                                    |
| `sepia`              | `filter`          | none                                                    |
| `backdropFilter`     | `backdrop-filter` | none                                                    |
| `backdropBlur`       | `backdrop-filter` | [blurs](/styled-system/theming/default-theme#blurs)     |
| `backdropBrightness` | `backdrop-filter` | none                                                    |
| `backdropContrast`   | `backdrop-filter` | none                                                    |
| `backdropDropShadow` | `backdrop-filter` | [shadows](/styled-system/theming/default-theme#shadows) |
| `backdropGrayscale`  | `backdrop-filter` | none                                                    |
| `backdropHueRotate`  | `backdrop-filter` | none                                                    |
| `backdropInvert`     | `backdrop-filter` | none                                                    |
| `backdropSaturate`   | `backdrop-filter` | none                                                    |
| `backdropSepia`      | `backdrop-filter` | none                                                    |

:::note status=warning
When using `blur`, `brightness`, `backdropBlur`, `backdropBrightness`, etc., you need to set `filter` and `backdropFilter` to `auto`.
:::
