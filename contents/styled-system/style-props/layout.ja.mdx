---
title: レイアウト
description: Yamada UIのStyle propsについて学ぶ。Style propsは、コンポーネントにpropsを渡すだけでコンポーネントのスタイルを変更する方法です。
order: 4
---

```tsx live=false
// When to use theme token value.
<Box w='md' />

// When to use custom value.
<Box w='1rem' />

// When to use responsive styles.
<Box w={{ base: 'lg', md: '1rem' }} />

// When to use color mode styles.
<Box w={['md', 'sm']} />
```
