title: エフェクト
description: Yamada UIのStyle propsについて学ぶ。Style propsは、コンポーネントにpropsを渡すだけでコンポーネントのスタイルを変更する方法です。
order: 8
---

```tsx live=false
// When to use theme token value.
<Box shadow='md' />

// When to use custom value.
<Box shadow='0 0 0 1px rgba(0, 0, 0, 0.05)' />

// When to use responsive styles.
<Box shadow={{ base: 'lg', md: 'md' }} />

// When to use color mode styles.
<Box shadow={['md', 'dark-md']} />
```

| Prop                                 | CSS Property            | Theme Key                                                    |
| ------------------------------------ | ----------------------- | ------------------------------------------------------------ |
| `shadow`, `boxShadow`                | `box-shadow`            | [shadows](/docs/styled-system/theming/default-theme/shadows) |
| `textShadow`                         | `text-shadow`           | [shadows](/docs/styled-system/theming/default-theme/shadows) |
| `blendMode`, `mixBlendMode`          | `mix-blend-mode`        | none                                                         |
| `bgBlendMode`, `backgroundBlendMode` | `background-blend-mode` | none                                                         |
| `isolation`                          | `isolation`             | none                                                         |
| `opacity`                            | `opacity`               | none                                                         |
| `filter`                             | `filter`                | none                                                         |
| `backdropFilter`                     | `backdrop-filter`       | none                                                         |
