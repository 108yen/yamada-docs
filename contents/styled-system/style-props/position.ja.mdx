---
title: ポジション
description: Yamada UIのStyle propsについて学ぶ。Style propsは、コンポーネントにpropsを渡すだけでコンポーネントのスタイルを変更する方法です。
order: 5
---

```tsx live=false
// When to use theme token value.
<Box pos="absolute" top='md' />

// When to use custom value.
<Box pos="absolute" top='2rem' />

// When to use responsive styles.
<Box pos="absolute" top={{ base: 'md', md: '8px' }} />

// When to use color mode styles.
<Box pos="absolute" top={['md', 'sm']} />
```

| Prop                             | CSS Property         | Theme Key                                                       |
| -------------------------------- | -------------------- | --------------------------------------------------------------- |
| `pos`, `position`                | `position`           | none                                                            |
| `zIndex`                         | `z-index`            | [zIndices](/docs/styled-system/theming/default-theme/z-indices) |
| `inset`                          | `inset`              | none                                                            |
| `top`                            | `top`                | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `left`                           | `left`               | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `bottom`                         | `bottom`             | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `right`                          | `right`              | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetY`                         | `top` + `bottom`     | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetX`                         | `left` + `right`     | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetInline`                    | `inset-inline`       | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetStart`, `insetInlineStart` | `inset-inline-start` | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetEnd`, `insetInlineEnd`     | `inset-inline-end`   | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetBlock`                     | `inset-block`        | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetBlockStart`                | `inset-block-start`  | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
| `insetBlockEnd`                  | `inset-block-end`    | [spaces](/docs/styled-system/theming/default-theme/spaces)      |
