## Installation

Let's create the application.

Execute one of the following commands in the terminal.

<PackageManagers

  }}
/>

## Add `UIProvider`

After installing Yamada UI, add the `UIProvider`.

```tsx live=false title=index.tsx highlight={10,12}
import { createRoot } from 'react-dom/client'
import { App } from './app'
import { UIProvider } from '@yamada-ui/react'

const container = document.getElementById('app')

const root = createRoot(container!)

root.render(
  <UIProvider>
    <App />
  </UIProvider>,
)
```

:::note
The [Default theme](/styled-system/theming/default-theme) is included inside the `UIProvider`.
:::

:::note
If you want to customize the theme or configuration, please check [Customize the theme](/styled-system/theming/customize-theme) and [Customize the configuration](/styled-system/configure/customize-config).
:::

## Use the components

Once you've added the `UIProvider`, call the components within the application.

```tsx live=false title=app.tsx
import { FC } from 'react'
import { Button } from '@yamada-ui/react'

export const App: FC = () => {
  return <Button>Click me!</Button>
}
```
