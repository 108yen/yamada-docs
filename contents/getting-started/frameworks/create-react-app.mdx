## Install the package

First, create an application.

Run one of the following commands in the terminal.

<PackageManagers
  packageNameOrCommand={{
    pnpm: "pnpm create react-app my-app --template typescript",
    npm: "npx create-react-app my-app --template typescript",
    yarn: "yarn create react-app my-app --template typescript",
  }}
/>

With Yamada UI, you can use most components and hooks just by installing `@yamada-ui/react`.

## Adding `UIProvider`

After installing Yamada UI, add `UIProvider`.

```tsx live=false title=index.tsx highlight={10,12}
import { createRoot } from "react-dom/client"
import { App } from "./app"
import { UIProvider } from "@yamada-ui/react"

const container = document.getElementById("app")

const root = createRoot(container!)

root.render(
  <UIProvider>
    <App />
  </UIProvider>,
)
```

:::note
The [default theme](/styled-system/theming/default-theme) is included in `UIProvider`.
:::

:::note
If you want to customize the theme or config, please check [customize the theme](/styled-system/theming/customize-theme) and [customize the config](/styled-system/configure/customize-config).
:::

## Using the Component

Once you've added `UIProvider`, call the component within your application.

```tsx live=false title=app.tsx
import { FC } from "react"
import { Button } from "@yamada-ui/react"

export const App: FC = () => {
  return <Button>Click me!</Button>
}
```
