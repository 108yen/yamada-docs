## パッケージをインストールする

まず、アプリケーションを作成します。

ターミナルで次のコマンドのいずれかを実行します。

<PackageManagers
  packageNameOrCommand={{
    pnpm: 'pnpm create react-app my-app --template typescript',
    npm: 'npx create-react-app my-app --template typescript',
    yarn: 'yarn create react-app my-app --template typescript',
  }}
/>

Yamada UIは、`@yamada-ui/react`をインストールするだけで、ほとんどのコンポーネントやフックを使用できます。

<PackageManagers packageNameOrCommand='@yamada-ui/react' />

テーブル・カレンダー・カルーセルなど使用する場合は、別途インストールをする必要があります。

## `UIProvider`を追加する

Yamada UIをインストール後、`UIProvider`を追加します。

```tsx live=false title=index.tsx
import { createRoot } from 'react-dom/client'
import { App } from './app'
import { UIProvider } from '@yamada-ui/react'

const container = document.getElementById('app')

const root = createRoot(container!)

root.render(
  <UIProvider>
    <App />
  </UIProvider>,
)
```

<Alert status='warning'>
  <AlertIcon />
  [デフォルトのテーマ](/docs/styled-system/theming/default-theme)は、`UIProvider`のなかに含まれています。
</Alert>

<Alert status='info'>
  <AlertIcon />
  独自のテーマやコンフィグの作成は、[こちら](/docs/styled-system/theming/create-theme)をご覧ください。
</Alert>

## コンポーネントを使用する

`UIProvider`を追加したら、アプリケーション内でコンポーネントを呼び出します。

```tsx live=false title=app.tsx
import { FC } from 'react'
import { Button } from '@yamada-ui/react'

export const App: FC = () => {
  return <Button>Click me!</Button>
}
```
