## Install the package

First, create an application.

Run one of the following commands in the terminal.

## Add `UIProvider`, `CacheProvider`, and `ColorModeScript`

In Next.js 13, `server components` are used by default. However, **Yamada UI is a `client component`**.

Pages and components that use Yamada UI components need to add `'use client'` and convert to `client components`.

After installing Yamada UI, add `UIProvider`, `CacheProvider`, and `ColorModeScript`.

```tsx live=false title=layout.tsx highlight={10,12,13,14}
"use client"

import { colorModeManager, UIProvider, ColorModeScript } from "@yamada-ui/react"
import { CacheProvider } from "@yamada-ui/nextjs"

const { cookieStorageManager } = colorModeManager

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <ColorModeScript type="cookie" nonce="testing" />

        <CacheProvider>
          <UIProvider colorModeManager={cookieStorageManager}>{children}</UIProvider>
        </CacheProvider>
      </body>
    </html>
  )
}
```

:::note status=warning
You need to add `'use client'` at the beginning of the page.
:::

## Using the Component

Once you've added `UIProvider`, you call the component within the application.

```tsx live=false title=page.tsx
"use client"

import { Button } from "@yamada-ui/react"

export default function Home() {
  return <Button>Click me!</Button>
}
```

:::note status=warning
You need to add `'use client'` at the beginning of the page.
:::
