## パッケージをインストールする

まず、アプリケーションを作成します。

ターミナルで次のコマンドのいずれかを実行します。

<PackageManagers
  packageNameOrCommand={{
    pnpm: "pnpm create next-app my-app --typescript",
    npm: "npx create-next-app my-app --typescript",
    yarn: "yarn create next-app my-app --typescript",
  }}
/>

Yamada UIは、`@yamada-ui/react`と`@yamada-ui/nextjs`をインストールするだけで、ほとんどのコンポーネントやフックを使用できます。

<PackageManagers packageNameOrCommand={["@yamada-ui/react", "@yamada-ui/nextjs"]} />

テーブル・カレンダー・カルーセルなど使用する場合は、別途インストールをする必要があります。

## `UIProvider`・`CacheProvider`・`ColorModeScript`を追加する

Next.js 13では、デフォルトで`サーバーコンポーネント`を使用しています。しかし、**Yamada UIは、`クライアントコンポーネント`です**。

Yamada UIのコンポーネントを使用するページやコンポーネントは、`'use client'`を追加して、`クライアントコンポーネント`に変換する必要があります。

Yamada UIをインストール後、`UIProvider`と`CacheProvider`と`ColorModeScript`を追加します。

```tsx live=false title=layout.tsx highlight={10,12,13,14}
"use client"

import { colorModeManager, UIProvider, ColorModeScript } from "@yamada-ui/react"
import { CacheProvider } from "@yamada-ui/nextjs"

const { cookieStorageManager } = colorModeManager

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en">
      <body>
        <ColorModeScript type="cookie" nonce="testing" />

        <CacheProvider>
          <UIProvider colorModeManager={cookieStorageManager}>{children}</UIProvider>
        </CacheProvider>
      </body>
    </html>
  )
}
```

:::note status=warning
`'use client'`を、ページの先頭に追加する必要があります。
:::

## コンポーネントを使用する

`UIProvider`を追加したら、アプリケーション内でコンポーネントを呼び出します。

```tsx live=false title=page.tsx
"use client"

import { Button } from "@yamada-ui/react"

export default function Home() {
  return <Button>Click me!</Button>
}
```

:::note status=warning
`'use client'`を、ページの先頭に追加する必要があります。
:::
