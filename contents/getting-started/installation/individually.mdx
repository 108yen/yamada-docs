## Install the Package

Install the core and utilities of Yamada UI.

Run one of the following commands in the terminal.

<PackageManagers
  packageNameOrCommand={['@yamada-ui/core', '@yamada-ui/theme', '@yamada-ui/utils']}
/>

Next, install the components and hooks you want to use in your project.

Here, let's try installing `@yamada-ui/layouts` which includes `Box`.

<PackageManagers packageNameOrCommand='@yamada-ui/layouts' />

:::note
The package names of components and hooks are listed on each documentation page.
:::

## Adding `ThemeProvider`

After installing the core and components, add `ThemeProvider` to the root of your application and pass the [default theme](/styled-system/theming/default-theme) and [default config](/styled-system/configure/default-config) from `@yamada-ui/theme`. If you have your own theme or config, pass those instead.

```tsx live=false highlight={6,7,8,11}
import { ThemeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { defaultTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ResetStyle />
      <GlobalStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

If you want to support dark mode, add `ColorModeProvider` as well.

```tsx live=false highlight={7,12}
import { ThemeProvider, ColorModeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { defaultTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ColorModeProvider config={defaultConfig}>
        <ResetStyle />
        <GlobalStyle />

        <YourApplication />
      </ColorModeProvider>
    </ThemeProvider>
  )
}
```

:::note
If you want to customize the theme or config, please check [customize the theme](/styled-system/theming/customize-theme) and [customize the config](/styled-system/configure/customize-config).
:::

:::note status=warning
If you do not pass `theme` or `config` to `ThemeProvider`, the styles and tokens of the components will not be applied.
:::

## Using the Component

Call the component within the application.

```tsx live=false
import { Box } from '@yamada-ui/layouts'

const Demo = () => {
  return <Box>This is Box</Box>
}
```

With this, the setup of Yamada UI is complete!
