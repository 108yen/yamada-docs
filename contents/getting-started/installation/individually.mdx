## Installation

Install the core and utilities of Yamada UI.

Run one of the following commands in the terminal.

<PackageManagers
  packageNameOrCommand={['@yamada-ui/core', '@yamada-ui/theme', '@yamada-ui/utils']}
/>

Next, install the components or hooks you want to use in the project.

For this example, let's install `@yamada-ui/layouts` which includes components like `Box`.

<PackageManagers packageNameOrCommand='@yamada-ui/layouts' />

:::note
The package names for components and hooks are mentioned on their respective documentation pages.
:::

## Add `ThemeProvider`

After installing the core and components, add the `ThemeProvider` to the root of the application, and pass the [Default theme](/styled-system/theming/default-theme) and [Default config](/styled-system/configure/default-config) from `@yamada-ui/theme`. If you have your own theme or config, use them instead.

```tsx live=false highlight={6,7,8,11}
import { ThemeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { defaultTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ResetStyle />
      <GlobalStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

If you want to support dark mode, add the `ColorModeProvider`.

```tsx live=false highlight={7,12}
import { ThemeProvider, ColorModeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { defaultTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ColorModeProvider config={defaultConfig}>
        <ResetStyle />
        <GlobalStyle />

        <YourApplication />
      </ColorModeProvider>
    </ThemeProvider>
  )
}
```

:::note
If you wish to customize the theme or config, please check [Customize the Theme](/styled-system/theming/customize-theme) and [Customize the Config](/styled-system/configure/customize-config).
:::

:::note status=warning
If you don't pass the `theme` or `config` to `ThemeProvider`, the styles and tokens of the components won't be applied.
:::

## use the components

Call the components within the application.

```tsx live=false
import { Box } from '@yamada-ui/layouts'

const Demo = () => {
  return <Box>This is Box</Box>
}
```

That's it! You've successfully set up Yamada UI!
