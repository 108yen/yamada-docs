## パッケージをインストールする

Yamada UIは、`@yamada-ui/react`をインストールするだけで、ほとんどのコンポーネントやフックを使用できます。

ターミナルで次のコマンドのいずれかを実行します。

<PackageManagers packageName='@yamada-ui/react' />

テーブル・カレンダー・カルーセルなど使用する場合は、別途インストールする必要があります。

<Alert status='warning'>
  <AlertIcon />
  Yamada UIは、React 18のみ互換性があります。React 17以前を使用している場合は、React 18までアップグレードしてください。
</Alert>

## `UIProvider`を追加する

Yamada UIをインストール後、アプリケーションのルートに`UIProvider`を追加します。

```tsx live=false
import * as React from 'react'
import { UIProvider } from '@yamada-ui/react'

const App = () => {
  return (
    <UIProvider>
      <YourApplication />
    </UIProvider>
  )
}
```
