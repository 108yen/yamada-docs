## Install the Packages

With Yamada UI, by just installing `@yamada-ui/react`, you can use most of the components and hooks.

Run one of the following commands in the terminal.

:::note status=warning
Yamada UI is only compatible with React 18. If you're using React 17 or earlier, please upgrade to React 18.
:::

## Add the `UIProvider`

After installing Yamada UI, add `UIProvider` to the root of the application.

```tsx live=false highlight={5,7}
import { UIProvider } from '@yamada-ui/react'

const App = () => {
  return (
    <UIProvider>
      <YourApplication />
    </UIProvider>
  )
}
```

## Using Components

Once you've added the `UIProvider`, call the components within the application.

```tsx live=false
import { Button } from '@yamada-ui/react'

const Demo = () => {
  return <Button>Click me!</Button>
}
```

That's it! You've successfully set up Yamada UI!
