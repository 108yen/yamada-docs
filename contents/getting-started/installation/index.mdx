## Install the Package

By installing `@yamada-ui/react`, you can use most of the components and hooks in Yamada UI.

Run one of the following commands in your terminal.

<PackageManagers packageNameOrCommand="@yamada-ui/react" />

If you want to use features like tables, calendars, carousels, etc., you need to install them separately.

:::note status=warning
Yamada UI is only compatible with React 18. If you are using React 17 or earlier, please upgrade to React 18.
:::

## Add `UIProvider`

After installing Yamada UI, add `UIProvider` to the root of your application.

```tsx live=false highlight={5,7}
import { UIProvider } from "@yamada-ui/react"

const App = () => {
  return (
    <UIProvider>
      <YourApplication />
    </UIProvider>
  )
}
```

## Using the Component

Once you've added the `UIProvider`, call the component within your application.

```tsx live=false
import { Button } from "@yamada-ui/react"

const Demo = () => {
  return <Button>Click me!</Button>
}
```

With this, the setup of Yamada UI is complete!
