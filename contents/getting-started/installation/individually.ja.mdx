## パッケージをインストールする

Yamada UIのコアやユーティリティをインストールします。

ターミナルで次のコマンドのいずれかを実行します。

<PackageManagers packageName={['@yamada-ui/core', '@yamada-ui/theme', '@yamada-ui/utils']} />

次に、プロジェクトに使用したいコンポーネントやフックをインストールしていきます。

ここでは、`Box`などを含んでいる`@yamada-ui/layouts`をインストールしてみましょう。

<PackageManagers packageName='@yamada-ui/layouts' />

<Alert status='info'>
  <AlertIcon />
  コンポーネントやフックのパッケージ名は、各ドキュメントページに記載されています。
</Alert>

## `ThemeProvider`を追加する

コアやコンポーネントをインストール後、アプリケーションのルートに`ThemeProvider`を追加し、`@yamada-ui/theme`から[デフォルトのテーマとコンフィグ](/docs/styled-system/theming/default-theme)を渡します。もし、あなたのテーマやコンフィグがある場合は、そちらを渡します。

```tsx live=false
import { ThemeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { defaultTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ResetStyle />
      <GlobalStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

ダークモードを対応させたい場合は、`ColorModeProvider`も追加します。

```tsx live=false
import { ThemeProvider, ColorModeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { defaultTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={defaultTheme} config={defaultConfig}>
      <ColorModeProvider config={defaultConfig}>
        <ResetStyle />
        <GlobalStyle />

        <YourApplication />
      </ColorModeProvider>
    </ThemeProvider>
  )
}
```

<Alert status='info'>
  <AlertIcon />
  独自のテーマやコンフィグの作成は、[こちら](/docs/styled-system/theming/create-theme)をご覧ください。
</Alert>

<Alert status='warning'>
  <AlertIcon />
  `theme`や`config`を`ThemeProvider`に渡さない場合、コンポーネントのスタイルやトークンは反映されません。
</Alert>

## コンポーネントを使用する

アプリケーション内でコンポーネントを呼び出します。

```tsx live=false
import { Box } from '@yamada-ui/layouts'

const Demo = () => {
  return <Box>This is Box</Box>
}
```

これで、Yamada UIのセットアップは完了です！
