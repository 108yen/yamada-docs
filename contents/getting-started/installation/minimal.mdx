## Using `ui`

After installation, call `ui` within the application.

`ui` is an object of JSX elements that have enabled the Yamada UI style system, and can also be used as a function for custom components to receive the Yamada UI style system.

Use the `ui.<element>` notation to generate basic HTML elements with a style system.
For example, to generate a `div` of plain HTML elements with a style system, write `<ui.div />`.

```tsx live=false
import { ui } from "@yamada-ui/core"

const Demo = () => {
  return (
    <ui.div>
      <ui.p>This is div</ui.p>
    </ui.div>
  )
}
```

To set the style, pass `props`.

Let's try it with `button` this time.

```tsx
<ui.button
  background="#ef4444"
  color="#ffffff"
  padding="1rem"
  borderRadius=".5rem"
  _hover={{ background: "#dc2626" }}
>
  Click me!
</ui.button>
```

:::note
If you want to learn more about `Style props`, please check [here](/styled-system/style-props).
:::

## Creating a Component

`ui` can be used in two ways: the JSX element `<ui.div />` and the function `ui('div')` that returns a JSX element.

The function is suitable for generating simple components.

```tsx live=false
import { ui } from "@yamada-ui/core"

const Button = ui("button")

const Demo = () => {
  return <Button>Click me!</Button>
}
```

You can also set default styles.

```tsx live=false
import { ui } from "@yamada-ui/core"

const Button = ui("button", {
  baseStyle: {
    background: "#ef4444",
    color: "#ffffff",
    padding: "1rem",
    borderRadius: ".5rem",
    _hover: { background: "#dc2626" },
  },
})

const Demo = () => {
  return <Button>Click me!</Button>
}
```

## If you want to use animation

Yamada UI provides many utilities such as `Motion`, a component specialized for animation, and `useAnimation`, which can be described like CSS's `keyframes`.

### Using `useAnimation`

Install `@yamada-ui/use-animation`.

<PackageManagers packageNameOrCommand="@yamada-ui/use-animation" />

```tsx functional=true
const animation = useAnimation({
  keyframes: {
    "0%": {
      bg: "#ef4444",
    },
    "20%": {
      bg: "#22c55e",
    },
    "40%": {
      bg: "#a855f7",
    },
    "60%": {
      bg: "#eab308",
    },
    "80%": {
      bg: "#3b82f6",
    },
    "100%": {
      bg: "#ef4444",
    },
  },
  duration: "10s",
  iterationCount: "infinite",
  timingFunction: "linear",
})

return <ui.div w="full" h="xs" animation={animation} />
```

### Using `Motion`

Install `@yamada-ui/motion`.

<PackageManagers packageNameOrCommand="@yamada-ui/motion" />

```tsx
<ui.div display="flex" placeContent="center" placeItems="center" w="100%" h="24rem">
  <Motion
    animate={{
      scale: [1, 2, 2, 1, 1],
      rotate: [0, 0, 180, 180, 0],
      borderRadius: ["0%", "0%", "50%", "50%", "0%"],
    }}
    transition={{
      duration: 2,
      ease: "easeInOut",
      times: [0, 0.2, 0.5, 0.8, 1],
      repeat: Infinity,
      repeatDelay: 1,
    }}
    w="7.5rem"
    h="7.5rem"
    bg="primary"
  />
</ui.div>
```

:::note
If you want to learn more about animation, please check [here](/styled-system/animation).
:::

## If you want to use tokens such as colors and spaces

If you want to use the [default theme](/styled-system/theming/default-theme) of Yamada UI, install `@yamada-ui/theme`.

<PackageManagers packageNameOrCommand="@yamada-ui/theme" />

After installing the theme, add `ThemeProvider` to the root of your application and pass the [default theme](/styled-system/theming/default-theme) and [default config](/styled-system/configure/default-config) from `@yamada-ui/theme`. If you have your own theme or config, pass them instead.

```tsx live=false highlight={6,7,8,11}
import { ThemeProvider, ResetStyle, GlobalStyle } from "@yamada-ui/core"
import { baseTheme, defaultConfig } from "@yamada-ui/theme"

const App = () => {
  return (
    <ThemeProvider theme={baseTheme} config={defaultConfig}>
      <ResetStyle />
      <GlobalStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

If you want to support dark mode, also add `ColorModeProvider`.

```tsx live=false highlight={7,12}
import { ThemeProvider, ColorModeProvider, ResetStyle, GlobalStyle } from "@yamada-ui/core"
import { baseTheme, defaultConfig } from "@yamada-ui/theme"

const App = () => {
  return (
    <ThemeProvider theme={baseTheme} config={defaultConfig}>
      <ColorModeProvider config={defaultConfig}>
        <ResetStyle />
        <GlobalStyle />

        <YourApplication />
      </ColorModeProvider>
    </ThemeProvider>
  )
}
```

:::note
If you want to customize the theme or config, please check [how to customize the theme](/styled-system/theming/customize-theme) and [how to customize the config](/styled-system/configure/customize-config).
:::

:::note status=warning
If you do not pass `theme` or `config` to `ThemeProvider`, the tokens will not be reflected.
:::
