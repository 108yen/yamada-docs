## パッケージをインストールする

Yamada UIのコアやユーティリティをインストールします。

ターミナルで次のコマンドのいずれかを実行します。

<PackageManagers packageNameOrCommand={['@yamada-ui/core', '@yamada-ui/utils']} />

## `ui`を使用する

インストール後、アプリケーション内で`ui`を呼び出します。

`ui`は、Yamada UIのスタイルシステムを有効にしたJSX要素のオブジェクトであり、カスタムコンポーネントがYamada UIのスタイルシステムを受け取るための関数としても使用できます。

もし、`div`の要素を作成したい場合は、`<ui.div />`と記述します。

```tsx live=false
import { ui } from '@yamada-ui/core'

const Demo = () => {
  return (
    <ui.div>
      <ui.p>This is div</ui.p>
    </ui.div>
  )
}
```

スタイルを設定するには、`props`を渡します。

今回は、`button`でやってみましょう。

```tsx
<ui.button
  background='#ef4444'
  color='#ffffff'
  padding='1rem'
  borderRadius='.5rem'
  _hover={{ background: '#dc2626' }}
>
  Click me!
</ui.button>
```

<Alert status='info'>
  <AlertIcon />
  `Style props`をもっと学びたい場合は、[こちら](/docs/styled-system/style-props)をご覧ください。
</Alert>

## コンポーネントを作る

`ui`は、JSX要素の`<ui.div />`と関数の`ui('div')`と2つの使い方ができます。

関数は、簡単なコンポーネントを作成するのに適しています。

```tsx live=false
import { ui } from '@yamada-ui/core'

const Button = ui('button')

const Demo = () => {
  return <Button>Click me!</Button>
}
```

デフォルトのスタイルを設定することもできます。

```tsx live=false
import { ui } from '@yamada-ui/core'

const Button = ui('button', {
  baseStyle: {
    background: '#ef4444',
    color: '#ffffff',
    padding: '1rem',
    borderRadius: '.5rem',
    _hover: { background: '#dc2626' },
  },
})

const Demo = () => {
  return <Button>Click me!</Button>
}
```

## アニメーションを使用したい場合

Yamada UIは、アニメーションに特化したコンポーネントの`Motion`やCSSの`keyframes`のように記述できる`useAnimation`など、多くのユーティリティを提供しています。

### `useAnimation`を使ってみましょう

`@yamada-ui/use-animation`をインストールします。

<PackageManagers packageNameOrCommand='@yamada-ui/use-animation' />

```tsx
function Demo() {
  const animation = useAnimation({
    keyframes: {
      '0%': {
        bg: '#ef4444',
      },
      '20%': {
        bg: '#22c55e',
      },
      '40%': {
        bg: '#a855f7',
      },
      '60%': {
        bg: '#eab308',
      },
      '80%': {
        bg: '#3b82f6',
      },
      '100%': {
        bg: '#ef4444',
      },
    },
    duration: '10s',
    iterationCount: 'infinite',
    timingFunction: 'linear',
  })

  return <ui.div w='full' h='xs' animation={animation} />
}
```

### `Motion`を使ってみましょう

`@yamada-ui/motion`をインストールします。

<PackageManagers packageNameOrCommand='@yamada-ui/motion' />

```tsx
function Demo() {
  return (
    <ui.div display='flex' placeContent='center' placeItems='center' w='100%' h='20rem'>
      <Motion
        animate={{
          scale: [1, 2, 2, 1, 1],
          rotate: [0, 0, 180, 180, 0],
          borderRadius: ['0%', '0%', '50%', '50%', '0%'],
        }}
        transition={{
          duration: 2,
          ease: 'easeInOut',
          times: [0, 0.2, 0.5, 0.8, 1],
          repeat: Infinity,
          repeatDelay: 1,
        }}
        w='7rem'
        h='7rem'
        bg='primary'
      />
    </ui.div>
  )
}
```

  アニメーションをもっと学びたい場合は、[こちら](/docs/styled-system/animation)をご覧ください。
</Alert>

## 色やスペースなどのトークンを使用したい場合

Yamada UIの[デフォルトのテーマ](/docs/styled-system/theming/default-theme)を使用したい場合は、`@yamada-ui/theme`をインストールします。

<PackageManagers packageNameOrCommand='@yamada-ui/theme' />

テーマをインストール後、アプリケーションのルートに`ThemeProvider`を追加し、`@yamada-ui/theme`から[デフォルトのテーマとコンフィグ](/docs/styled-system/theming/default-theme)を渡します。もし、あなたのテーマやコンフィグがある場合は、そちらを渡します。

```tsx live=false
import { ThemeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { baseTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={baseTheme} config={defaultConfig}>
      <ResetStyle />
      <GlobalStyle />

      <YourApplication />
    </ThemeProvider>
  )
}
```

ダークモードを対応させたい場合は、`ColorModeProvider`も追加します。

```tsx live=false
import { ThemeProvider, ColorModeProvider, ResetStyle, GlobalStyle } from '@yamada-ui/core'
import { baseTheme, defaultConfig } from '@yamada-ui/theme'

const App = () => {
  return (
    <ThemeProvider theme={baseTheme} config={defaultConfig}>
      <ColorModeProvider config={defaultConfig}>
        <ResetStyle />
        <GlobalStyle />

        <YourApplication />
      </ColorModeProvider>
    </ThemeProvider>
  )
}
```

<Alert status='info'>
  <AlertIcon />
  独自のテーマやコンフィグの作成は、[こちら](/docs/styled-system/theming/create-theme)をご覧ください。
</Alert>

<Alert status='warning'>
  <AlertIcon />
  `theme`や`config`を`ThemeProvider`に渡さない場合、トークンは反映されません。
</Alert>
