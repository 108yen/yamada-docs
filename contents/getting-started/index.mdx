Yamada UI offers a variety of components, hooks, themes, and utilities that are broken down depending on the size and case of the project.

- To install all components and hooks, please check [here](/getting-started/installation).
- To install components or hooks individually and improve performance, please check [here](/getting-started/installation/individually).
- If you only want to use the style system of Yamada UI, please check [here](/getting-started/installation/minimal).
