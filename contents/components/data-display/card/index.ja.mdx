## インポート

```ts
import { Card, CardHeader, CardBody, CardFooter } from "@yamada-ui/react"
```

- `Card`: カードをラッパーコンポーネントです。
- `CardHeader`: カードのヘッダーを表示するコンポーネントです。
- `CardBody`: カードのメインコンテンツを表示するコンポーネントです。
- `CardFooter`: カードのフッターを表示するコンポーネントです。

## 使い方

```tsx
<Card>
  <CardHeader>
    <Heading size="md">『ドラゴンボール』（DRAGON BALL）</Heading>
  </CardHeader>

  <CardBody>
    <Text>
      『ドラゴンボール』（DRAGON
      BALL）は、鳥山明による日本の漫画作品。『週刊少年ジャンプ』（集英社）にて1984年51号から1995年25号まで連載された。世界中に散らばった七つの球をすべて集めると、どんな願いも一つだけ叶えられるという秘宝・ドラゴンボールと、主人公・孫悟空（そん・ごくう）を中心に展開する、「冒険」「夢」「バトル」「友情」などを描いた長編漫画。
    </Text>
  </CardBody>
</Card>
```
