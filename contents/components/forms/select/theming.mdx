The `Select` is a [multi part component](/styled-system/theming/component-style#multi-part-component).

:::note
`Select` inherits the style from [NativeSelect](/components/forms/native-select).
:::

:::note
If you want to change the style of the component, please check [here](/styled-system/theming/customize-theme#changing-the-style-of-components).
:::
